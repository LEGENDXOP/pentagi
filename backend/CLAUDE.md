# PentAGI Backend — AI Agent Navigation Hub

## Project Overview
PentAGI is an autonomous AI-driven penetration testing platform. The Go backend orchestrates multi-agent workflows (flows) that plan, execute, and report security assessments. Agents use LLM providers, Docker containers, and specialized tools (browser, terminal, nuclei, search engines) to perform tasks.

**Architecture:** HTTP/GraphQL → Server → Controller → Provider → Performer → Tools → External Services

## Architecture Summary
1. **HTTP/GraphQL** layer receives requests (pkg/server/, pkg/graph/)
2. **FlowController** (pkg/controller/flows.go) creates FlowWorkers with Docker containers
3. **FlowWorker** (pkg/controller/flow.go) spawns TaskWorkers → SubtaskWorkers
4. **FlowProvider** (pkg/providers/provider.go) connects to LLM and calls performAgentChain
5. **performAgentChain** (pkg/providers/performer.go:205) loops: LLM call → tool execution → check completion
6. **Tools** (pkg/tools/) execute actions in containers, browsers, search engines
7. **MasterAgent** (pkg/masteragent/) supervises flows autonomously via periodic LLM analysis
